# 💰 Hesabu Zangu

> **Simamia Pesa Zako Kila Siku** — Personal Finance Tracker iliyoundwa Tanzania 🇹🇿

---

## 📱 Kuhusu App

Hesabu Zangu ni app ya Android iliyoundwa kwa Watanzania wote — mfanyabiashara, bodaboda, mama lishe, mwanafunzi, au mfanyakazi — kusaidia kurekodi mapato na matumizi kila siku bila internet.

---

## ✨ Features

- 💰 **Rekodi Mapato** — Aina 8 za mapato
- 💸 **Rekodi Matumizi** — Aina 12 za matumizi
- 📊 **Dashboard** — Salio, faida, na muhtasari wa leo
- 📋 **Historia** — Tazama na tafuta rekodi zote
- 📈 **Ripoti** — Charts, pie charts, na uchambuzi wa mwezi
- 📄 **PDF Download** — Ripoti nzuri ya kuchapisha
- 📊 **Excel Download** — Spreadsheet ya kina (sheets 3)
- 🔔 **Reminders** — Notification za asubuhi na jioni
- 📵 **100% Offline** — Haihitaji internet kabisa

---

## 🏗️ Tech Stack

| Teknolojia | Matumizi |
|---|---|
| Flutter | UI Framework |
| Hive | Local Database (Offline) |
| pdf package | PDF Generation |
| excel package | Excel Generation |
| fl_chart | Charts na Graphs |
| flutter_local_notifications | Daily Reminders |
| GitHub | Version Control |
| Codemagic | CI/CD & Builds |

**Gharama ya server: Sh. 0** 🎉

## 🚀 Jinsi ya Kuendesha

```bash
# Clone project
git clone https://github.com/JINA-LAKO/hesabu_zangu.git
cd hesabu_zangu

# Install packages
flutter pub get

# Run app
flutter run

# Build APK
flutter build apk --release
```
